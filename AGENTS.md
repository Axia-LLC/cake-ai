## Workspace Overview

This is a Cupcake-managed workspace with automated learning consolidation and knowledge management.

## Core Principles

1. **Never delete, always archive** - Move files to `z_archive/` instead of deleting
2. **Checkboxes are inviolate** - Must resolve all applicable unchecked items before proceeding
3. **Economy first** - Prioritize brevity, eliminate redundancy, maintain focus
4. **Atomic structure** - Each file, rule, and learning is self-contained
5. **Zero-context ready** - Any agent should be able to pick up where others left off

## Key Files

- `USERS.md` - Critical information about the user's preferences and context
- `LEARNINGS.md` - Workspace-level consolidated learnings
- `agent/counter-register.md` - Chat counter and consolidation schedule
- `my-learnings/learnings-primary.md` - Top learnings (30 max + overflow)
