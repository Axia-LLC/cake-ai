# Cupcake Design Project

**Created:** 2026-04-18  
**Topic:** Design and implementation of Cupcake workspace scaffolding system

## Project Overview

Cupcake is a personal workspace scaffolding package that sets up a complete development environment from a blank workspace. It provides:

- **Folder structure** for projects, skills, learnings, and agent infrastructure
- **Agent-memory system** for automated learning consolidation across chat sessions
- **Rules and hooks** for maintaining knowledge hygiene and context economy
- **Template system** for consistent project creation

## Key Components

1. **Folder Structure** - Organized workspace layout with templates
2. **Agent-Memory Program** - Counter-based learning consolidation system
3. **Rules System** - Atomic rules for AI behavior and governance
4. **Hook System** - Automated triggers for memory management

## Current Focus

Working through the design of the agent-memory system and folder structure to ensure:
- Atomic structure
- Adaptability  
- Economy (no context bloat)
- Usefulness
- Robustness

## Goals

- Finalize folder structure and naming conventions
- Design counter-based consolidation logic
- Define learning hierarchies and economy rules
- Specify hook triggers and scripts
- Identify baseline rules for Cupcake shipment
