# My Projects

This folder contains all active and archived project work.

## Structure

- `template/` - Template structure for new projects
- `YYYYMMDD-[topic-name]/` - Individual project folders (date-prefixed)

## Creating a New Project

New projects follow the template structure:
- `readme.md` - Project overview and context
- `scratchpad.md` - Temporary working document
- `learnings.md` - Permanent project-specific learnings
- `z_archive/` - Archived files (never delete, only archive)
